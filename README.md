#  Movie Explorer App

A modern movie search and watchlist web application that allows users to discover movies, save favorites, and decide what to watch next — all in a sleek dark-themed interface.



##  Features

###  Live Search

* Search movies in real-time using the OMDb API
* Instant results as the user types or clicks search
* Displays movie posters, titles, and release year

###  Watchlist Toggle

* Add or remove movies from your watchlist
* Visual indication of saved movies
* (Optional) Persist data using local storage

###  Empty States

* User-friendly UI when:

  * No search results are found
  * Watchlist is empty

###  UI/UX Design

* Dark theme inspired by Disney+ and HBO Max
* Clean, responsive layout
* Sticky sidebar to view saved movies in real-time

###  Random Movie Night 

* Randomly selects a movie from your watchlist
* Helps users decide what to watch quickly



##  Tech Stack

* HTML
* CSS
* JavaScript


## APIs Used

* OMDb API → Movie search and basic details
* TMDB API → Additional metadata (optional)
* Watchmode API → Streaming availability (optional)



##  Core Functionality

* Fetch movie data using API calls
* Render results dynamically
* Manage watchlist state
* Handle empty states gracefully
* Provide smooth UI interactions
